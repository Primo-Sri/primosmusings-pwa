Who is Primo?

A lifelong learner driven by a trifecta of passions: writing, music, and sport. What began as a hobby project to document the Australian Open has evolved into Primosmusings — a comprehensive podcast where global sports cultures collide.

The Cricket Roots

My journey with sports started on the cricket pitch in India. My move to the US opened a new chapter — as a proud UConn Husky alumnus, I became a devoted follower of the New England Patriots and the Boston Red Sox.

❓ Help & FAQ

Seeing old content? App not updating? This page walks you through clearing cache on every device and browser — plus answers common questions about the app.

Why Does the App Look Outdated?

💾
What is a cache and why does it get stuck? ›

Your browser saves a local copy of this app so it loads fast — even offline. That copy is called the cache.

When Primo updates the app, your browser sometimes keeps serving the old saved copy instead of downloading the new one. The fix is simple: clear that saved copy and let the browser fetch fresh content.

The app now auto-updates — when a new version is deployed, it silently refreshes in the background. But if you're seeing stale content, the steps below will fix it immediately.

Clear Cache — Step by Step

🌐
Microsoft Edge — Windows & Mac ›
🌐 Edge on Windows
  1. 1Press Ctrl + Shift + Delete to open Clear browsing data
  2. 2Set Time range to All time
  3. 3Tick Cached images and files (untick Browsing history if you want to keep it)
  4. 4Click Clear now
  5. 5Come back to the app tab and press F5 or Ctrl + R
🌐 Edge on Mac
  1. 1Press ⌘ Cmd + Shift + Delete
  2. 2Set Time range to All time, tick Cached images and files
  3. 3Click Clear now, then refresh the tab
If you have the app installed as a PWA from Edge, uninstall it from the taskbar (right-click the app icon → Uninstall), then reinstall from the browser.
🔵
Google Chrome — Windows & Mac ›
🔵 Chrome on Windows
  1. 1Press Ctrl + Shift + Delete
  2. 2Set Time range to All time
  3. 3Tick Cached images and files
  4. 4Click Clear data
  5. 5Refresh the tab with Ctrl + Shift + R (hard reload)
🔵 Chrome on Mac
  1. 1Press ⌘ Cmd + Shift + Delete
  2. 2Set range to All time, tick Cached images and files, click Clear data
  3. 3Hard reload with ⌘ Cmd + Shift + R
Faster option: Click the 🔒 lock icon in the address bar → Site settings → Clear data. This clears only this site's cache, nothing else.
🤖
Android — Chrome App ›
🤖 Android · In Chrome
  1. 1Tap the three dots ⋮ menu in Chrome
  2. 2Tap Settings → Privacy and security → Clear browsing data
  3. 3Set time range to All time, tick Cached images and files
  4. 4Tap Clear data
  5. 5Return to the app and pull down to refresh
🤖 Android · Installed PWA
  1. 1Go to Settings → Apps → Chrome (or search "Primosmusings" in Apps)
  2. 2Tap Storage → Clear Cache
  3. 3Reopen the app
If it's still stale, long-press the app icon → App info → Clear Cache. This is the most direct route on Android.
🍎
iPhone & iPad — Safari ›
🍎 iOS · Safari browser
  1. 1Open the iPhone/iPad Settings app (not Safari settings)
  2. 2Scroll down and tap Apps → Safari
  3. 3Tap Clear History and Website Data
  4. 4Confirm, then reopen Safari and visit the app
Note: This clears all Safari history and data, not just this app. If you want to clear only this site, use the steps below instead.
🍎 iOS · Site-specific only
  1. 1Open Settings → Apps → Safari → Advanced → Website Data
  2. 2Search for primosmusings
  3. 3Swipe left on the entry and tap Delete
🍎 iOS · Installed PWA (home screen icon)
  1. 1Long-press the Primosmusings icon on your home screen
  2. 2Tap Remove App → Delete App
  3. 3Open Safari, go to the site, tap Share → Add to Home Screen to reinstall
iOS is the strictest about caching. If nothing else works, the delete + reinstall from home screen always fixes it.
🧭
Safari — Mac ›
🧭 Safari on Mac
  1. 1In the menu bar, click Safari → Settings (or Preferences on older macOS)
  2. 2Click the Privacy tab
  3. 3Click Manage Website Data…
  4. 4Search for primosmusings, select it, click Remove
  5. 5Close settings and reload the page with ⌘ Cmd + R
Quick option: In Safari, hold Shift while clicking the Reload button — this does a hard refresh that bypasses the cache without clearing everything.
🦊
Firefox — Windows & Mac ›
🦊 Firefox
  1. 1Press Ctrl + Shift + Delete (Windows) or ⌘ + Shift + Delete (Mac)
  2. 2Set time range to Everything
  3. 3Tick Cache
  4. 4Click Clear Now
  5. 5Hard reload with Ctrl + Shift + R

Quick Hard-Reload Shortcuts

Skip clearing cache entirely — a hard reload bypasses the cache for just this page load.

Windows / Linux (Chrome, Edge, Firefox) Ctrl+Shift+R
Mac (Chrome, Edge, Firefox) ⌘+Shift+R
Mac Safari Shift+Click Reload
Any browser (alternate) F5orCtrl+F5

App Questions

🎙
Episodes aren't loading — what do I do? ›
  1. 1Check your internet connection — the episode list loads live from Spotify/Anchor
  2. 2If connected, try a hard reload (Ctrl + Shift + R)
  3. 3If still blank, the proxy that fetches episodes may be temporarily down — wait a few minutes and try again
  4. 4As a fallback, tap Open on Spotify — that always works
📲
How do I install this app on my device? ›
🤖 Android
  1. 1Open the site in Chrome
  2. 2Tap the three dots ⋮ → Add to Home Screen
  3. 3Tap Install — the app appears on your home screen like a native app
🍎 iPhone / iPad
  1. 1Open the site in Safari (must be Safari, not Chrome)
  2. 2Tap the Share button (box with arrow pointing up)
  3. 3Tap Add to Home Screen → Add
💻 Desktop (Chrome / Edge)
  1. 1Look for the install icon (⊕ or computer icon) in the address bar
  2. 2Click it → Install
  3. 3The app opens in its own window, separate from the browser
📵
Does the app work offline? ›

Yes — partially. The app shell (layout, guest list, navigation) loads from your device's cache when offline.

The episode list and blog posts require an internet connection because they're loaded live from Spotify and the blog feed.

To actually listen offline, save episodes in the Spotify app directly — the Primosmusings website/PWA can't store audio files.
